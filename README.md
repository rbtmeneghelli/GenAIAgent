# GenAIAgent

<b>CONTEXTO</b>

O software foi desenvolvido para aplicar e armazenar conhecimentos importantes envolvendo IA (agentes, workflows e aprendizados de maquina)

Para desenvolvimento desse software foi escolhido as seguintes tecnologias:

  - C# com NET 10
  - Repositorio de Codigo >> GitHub
  - Frameworks >> Microsoft.Agent.AI e MLNET
  - Ferramentas de código >> Typescript, HTML 5 e SASS/Css
    
<b>MOTIVO DA DECISÃO</b>
  - Devido a aceleração do uso da IA em nosso dia a dia, decidi desenvolver esse projeto para que eu possa praticar e colher conhecimento sobre o assunto para uso em futuros projetos.
    
<b>LIMITAÇÕES</b>
  - É necessario realizar o cadastro para poder usufruir os recursos do framework Microsoft.Agent.AI
       
<b>CONSIDERAÇÕES FINAIS</b>
  - Esse projeto não é para uso profissional, apenas academico.

<b>PASSOS PARA UTILIZAR</b>
  - Utilizar uma IDE da sua escolha, recomendo a utilização do VISUAL STUDIO ou VISUAL STUDIO CODE
  - Utilizar o SDK do NET 10 ou superior
